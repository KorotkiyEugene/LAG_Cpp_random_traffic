Для компіляції та запуску високопродуктивної моделі wormhole МнК з агрегацією каналів необхідно:
1.	Встановити програму Cygwin версії 1.7;
2.	Встановити SystemC версії 2.2;
3.	Відкрити консоль Сygwin і перейти в директорію, що містить вихідний код моделі;
4.	Якщо є необхідність, змінити параметри моделі в файлі “parameters.h”;
5.	Для компіляції моделі виконати з консолі Сygwin команду:
g++ -I /usr/local/systemc-2.2/include -L /usr/local/systemc-2.2/lib-cygwin main.cpp -lsystemc -o run.x
В результаті компіляції буде створено виконавчий файл “run.x”; 
6.	Для початку процедури моделювання необхідно запустити на виконання файл “run.x”. Результати моделювання будуть виведені в консоль Сygwin і за 
необхідності можуть бути перенаправлені в файл.
